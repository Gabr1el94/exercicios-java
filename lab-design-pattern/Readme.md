###Singlegton 
É um tipo de projeto criacional, no qual faz parte em padrões de projeto com o objetivo
de garantir que apenas um objeto desse tipo exista e forneça um único ponto de acesso a ele para qualquer outro código.

###Strategy
É um projeto comportamental, cujo objetivo é simplificar a variação 
de algoritmos para a resolução de um mesmo problema; 

###Facade
É um projeto estrutural que prover uma interface que reduza a
complexidade nas integrações com subsistemas;
